# ECHOES Cloud Assessment Framework Wiki

This repository supports the companion wiki for the **ECHOES Cloud Assessment Framework**.

The wiki provides structured documentation on the Cloud Assessment Framework for the ECCCH ecosystem, including:

- ECCCH resource types and categories
- evaluation dimensions
- assessment tools and instruments
- indicators and related guidance
- implementation-oriented support materials

The actual published wiki can be found here:

**https://echoes-eccch-technical.github.io/cloud-assessment-framework**
