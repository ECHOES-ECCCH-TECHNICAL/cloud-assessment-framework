# ECCCH - Cloud Assessment Framework Documentation

!!! warning "This site is work in progress."

!!! info "This documentation is currently generated from the ECHOES' *Matrix of Evaluation Criteria* and may be incomplete and subject to change."
This wiki provides evolving documentation on the *ECCCH Components*, *Resource Categories*, *Evaluation Dimensions*, *Assessment Tools*, and related guidance that support the Cloud Assessment Framework. Its content will be progressively expanded and refined as ECHOES WP9 advances and as the ECCCH ecosystem matures.

In the near future, the wiki will also describe more concrete procedures for different types of resources, including how to use the recommended assessment tools and how to share the resulting outputs with ECHOES in a consistent and reusable way.

At present, the wiki mainly records the assessment tools identified so far. Only a smaller, curated subset of these tools is expected to support the actual evaluation processes of ECCCH resources.
