# Assessment Tools

An assessment instrument, or simply an assessment tool, is a software application, service, or methodology that facilitates the collection of measurements used to compute specific metrics or indicators. 

This section documents the assessment tools currently identified within the framework. Each tool collection is described through its own index page, and each individual tool is documented on a dedicated page.

- [Software quality tools](software_quality_tools/index.md) — Software quality related tools.
- [Software functional tools](software_functional_tools/index.md) — Functional testing or validation tools.
- [Software E2E testing tools](software_e2e_testing_tools/index.md) — End-to-end testing tools.
- [Software static analysis tools](software_static_analysis_tools/index.md) — Static analysis tools.
- [Software performance tools](software_performance_tools/index.md) — Performance analysis tools.
- [Software security tools](software_security_tools/index.md) — Security-related tools.
- [Ontology tools](ontology_tools/index.md) — Ontology-oriented assessment or support tools.
- [KG tools](kg_tools/index.md) — Knowledge graph and data-oriented tools.
