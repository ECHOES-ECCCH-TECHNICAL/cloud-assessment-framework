# Software static analysis tools

Static analysis tools.

## Tools

- [Semgrep](./semgrep.md) — Multi-language static analysis tool with customizable rules.
- [SonarQube (Community Edition)](./sonarqube-community-edition.md) — Comprehensive code quality and security analysis platform. Community edition does not cover all languages/features.
- [CodeQL](./codeql.md) — Semantic code analysis engine developed by GitHub; focuses on security vulnerabilities.
- [Lizard](./lizard.md) — Cyclomatic Complexity Analyzer for many programming languages includingC#, C/C++, Java, JavaScript, Python and TypeScript.
- [PMD](./pmd.md) — Static analyzer for Java and other languages (includes JS support).
- [cppcheck](./cppcheck.md) — Static analysis tool for C and C++ detecting bugs and undefined behavior.
- [Clang-Tidy](./clang-tidy.md) — Part of LLVM/Clang project; performs linting and static analysis for C/C++/Objective-C.
- [Infer](./infer.md) — Static analysis tool from Meta for C, C++, Java, and Objective-C focused on bug detection.
- [OCLint](./oclint.md) — Analyzes C, C++, and Objective-C code for potential issues and style violations.
