# Europeana methodology for validating enrichments

*Collection: KG tools*

- **ID:** `KG_TOOLS_EUROPEANA_METHODOLOGY_FOR`
- **Dimensions covered:** Factual; Semantic; Context and Relevance; Completeness.
- **Description:** Methodology used by the Europeana Foundation to validate metadata enrichment efforts before acceptance. It combines documentation of the enrichment context, representative sampling, manual or semi-automated assessment of sample outputs, validation against agreed quality requirements, and, where relevant, calibration of confidence thresholds and iterative improvement of the enrichment process.

- **Quality Category:** Intrinsic; Contextual.
- **Automation:** semi-automatic
