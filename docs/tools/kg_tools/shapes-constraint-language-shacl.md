# Shapes Constraint Language (SHACL)

*Collection: KG tools*

- **ID:** `KG_TOOLS_SHAPES_CONSTRAINT_LANGUAGE`
- **Dimensions covered:** Consistency; Format.
- **Description:** A W3C language for validating RDF graphs against a set of conditions. These conditions are provided as shapes and other constructs expressed in the form of an RDF graph.
- **Link:** <a href="https://www.w3.org/TR/shacl/" target="_blank" rel="noopener noreferrer">https://www.w3.org/TR/shacl/</a>
- **Quality Category:** Intrinsic; Representational
