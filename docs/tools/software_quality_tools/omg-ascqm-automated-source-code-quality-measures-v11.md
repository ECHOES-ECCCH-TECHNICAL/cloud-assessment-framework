# OMG ASCQM (Automated Source Code Quality Measures) v1.1

*Collection: Software quality tools*

- **ID:** `SOFTWARE_QUALITY_TOOLS_OMG_ASCQM_AUTOMATED`
- **Dimensions covered:** Maintainability; Reliability; Performance Efficiency; Security.
- **Description:** OMG specification for automatically measuring software structural quality through static source-code analysis. It defines a catalog of weaknesses and detection patterns; scores per quality dimension are computed by counting and aggregating occurrences of these weaknesses by quality characteristic. The specification also defines conformance requirements to ensure that measurements are automated, objective, transparent, and repeatable.
- **Link:** <a href="https://www.omg.org/spec/ASCQM/1.1/PDF" target="_blank" rel="noopener noreferrer">https://www.omg.org/spec/ASCQM/1.1/PDF</a>
- **Adoption Likelihood:** High
- **Type of tool:** OMG specification; automated source-code structural quality measurement model (static-analysis-based)
