# SPDX specification

*Collection: Software quality tools*

- **ID:** `SOFTWARE_QUALITY_TOOLS_SPDX_SPECIFICATION`
- **Dimensions covered:** Security, Maintainability, Compatibility, Reliability.
- **Description:** SPDX is a standardized format for describing the components of a software system. It provides a common way for development teams to document what is included in their software, typically in the form of a Software Bill of Materials (SBOM).
