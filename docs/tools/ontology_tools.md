# Ontology Tools

Ontology-oriented assessment or support tools.

## Summary

- **Protégé Desktop** — Free, open-source OWL ontology editor with built-in integration to DL reasoners and support for debugging inconsistencies and refactoring ontologies.
- **WebProtégé** — Web-based collaborative ontology development environment supporting OWL 2, permissions, discussions, and change tracking for team-based ontology maintenance.
- **OOPS! (OntOlogy Pitfall Scanner!)** — Web-based ontology evaluation tool that detects common ontology “pitfalls” that can lead to modeling errors and quality issues.
- **HermiT OWL Reasoner** — OWL 2 DL reasoner that checks ontology consistency and computes class hierarchies (classification) and related inferences.
- **ELK Reasoner** — High-performance OWL 2 EL reasoner for large ontologies; used to classify and validate profile-consistent ontologies efficiently.
- **Openllet (Pellet-derived) Reasoner** — OWL 2 reasoner (Pellet lineage) for checking ontology consistency, classification, and inference services.
- **ROBOT** — Command-line tool (OBO/OWL-focused) for automating ontology workflows (e.g., validation steps, reports, transforms) commonly used in ontology engineering pipelines.
- **OntoMetrics** — Web-based tool that validates and displays statistics/metrics about a given ontology (upload OWL/RDF and obtain metric summaries).
- **Ontospy** — Python library/CLI for inspecting and visualizing RDF/OWL vocabularies, extracting ontology structure and generating lightweight documentation.
- **WIDOCO** — Wizard to generate enriched, customizable ontology documentation (terms, sections, visualizations) for publishing and reuse.
- **Pletka** — Pletka is a platform for collaboratively creating, documenting, and sharing semantic data models.
