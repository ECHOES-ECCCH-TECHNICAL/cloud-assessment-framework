# OntoMetrics

*Collection: Ontology tools*

- **ID:** `ONTOLOGY_TOOLS_ONTOMETRICS`
- **Dimensions covered:** Comprehensiveness; Sustainability.
- **Description:** Web-based tool that validates and displays statistics/metrics about a given ontology (upload OWL/RDF and obtain metric summaries).
- **Link:** <a href="https://ontometrics.informatik.uni-rostock.de/" target="_blank" rel="noopener noreferrer">https://ontometrics.informatik.uni-rostock.de/</a>
- **Main assessment use:** Ontology statistics; size and complexity profiling; metric-based summaries
