# Framework for Ontology Quality Assessment (FOCA)

*Collection: Ontology tools*

- **ID:** `ONTOLOGY_TOOLS_FRAMEWORK_FOR_ONTOLOGY`
- **Dimensions covered:** Comprehensiveness; Usability and Understandability; Interoperability and Compatibility; Sustainability.
- **Description:** Methodology for ontology evaluation based on a Goal–Question–Metric approach. It includes a questionnaire and a scoring model to assess ontology quality beyond formal consistency, with questions about completeness, reuse, documentation, organizational fitness, and alignment.
- **Link:** <a href="https://arxiv.org/abs/1612.03353" target="_blank" rel="noopener noreferrer">https://arxiv.org/abs/1612.03353</a>
- **Main assessment use:** Questionnaire-based and score-based ontology quality assessment beyond formal validation
