# OOPS! (OntOlogy Pitfall Scanner!)

*Collection: Ontology tools*

- **ID:** `ONTOLOGY_TOOLS_OOPS_ONTOLOGY_PITFALL`
- **Dimensions covered:** Usability and Understandability; Interoperability and Compatibility; Sustainability.
- **Description:** Web-based ontology evaluation tool that detects common ontology “pitfalls” that can lead to modeling errors and quality issues.
- **Link:** <a href="https://oops.linkeddata.es/" target="_blank" rel="noopener noreferrer">https://oops.linkeddata.es/</a>
- **Main assessment use:** Detection of common modeling pitfalls; ontology quality diagnostics; identification of best-practice violations
