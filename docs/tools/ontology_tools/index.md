# Ontology tools

Ontology-oriented assessment or support tools.

## Tools

- [Protégé Desktop](./protege-desktop.md) — Free, open-source OWL ontology editor with built-in integration to DL reasoners and support for debugging inconsistencies and refactoring ontologies.
- [WebProtégé](./webprotege.md) — Web-based collaborative ontology development environment supporting OWL 2, permissions, discussions, and change tracking for team-based ontology maintenance.
- [OOPS! (OntOlogy Pitfall Scanner!)](./oops-ontology-pitfall-scanner.md) — Web-based ontology evaluation tool that detects common ontology “pitfalls” that can lead to modeling errors and quality issues.
- [HermiT OWL Reasoner](./hermit-owl-reasoner.md) — OWL 2 DL reasoner that checks ontology consistency and computes class hierarchies (classification) and related inferences.
- [ELK Reasoner](./elk-reasoner.md) — High-performance OWL 2 EL reasoner for large ontologies; used to classify and validate profile-consistent ontologies efficiently.
- [Openllet (Pellet-derived) Reasoner](./openllet-pellet-derived-reasoner.md) — OWL 2 reasoner (Pellet lineage) for checking ontology consistency, classification, and inference services.
- [OntoMetrics](./ontometrics.md) — Web-based tool that validates and displays statistics/metrics about a given ontology (upload OWL/RDF and obtain metric summaries).
- [Ontospy](./ontospy.md) — Python library/CLI for inspecting and visualizing RDF/OWL vocabularies, extracting ontology structure and generating lightweight documentation.
- [WIDOCO](./widoco.md) — Wizard to generate enriched, customizable ontology documentation (terms, sections, visualizations) for publishing and reuse.
- [Pletka](./pletka.md) — Pletka is a platform for collaboratively creating, documenting, and sharing semantic data models.
- [Framework for Ontology Quality Assessment (FOCA)](./framework-for-ontology-quality-assessment-foca.md) — Methodology for ontology evaluation based on a Goal–Question–Metric approach. It includes a questionnaire and a scoring model to assess ontology quality beyond formal consistency, with questions about completeness, reuse, documentation, organizational fitness, and alignment.
- [Ontology Usability Scale (OUS)](./ontology-usability-scale-ous.md) — A questionnaire-style usability instrument proposed specifically for ontologies, with short statements intended to assess ontology usability in terms of effectiveness, efficiency, satisfaction, contextual relevance, maturity, intended use, and fitness for use. It is especially suitable for expert/user evaluation workshops or pilot-based assessment.
- [QASAR](./qasar.md) — Recent ontology quality-assurance platform that integrates multiple QA methodologies, interactive reports, and guided recommendations to support broader ontology evaluation.
