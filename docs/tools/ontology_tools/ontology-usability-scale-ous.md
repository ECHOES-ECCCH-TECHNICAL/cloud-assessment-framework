# Ontology Usability Scale (OUS)

*Collection: Ontology tools*

- **ID:** `ONTOLOGY_TOOLS_ONTOLOGY_USABILITY_SCALE`
- **Dimensions covered:** Usability and Understandability; Usage and Impact.
- **Description:** A questionnaire-style usability instrument proposed specifically for ontologies, with short statements intended to assess ontology usability in terms of effectiveness, efficiency, satisfaction, contextual relevance, maturity, intended use, and fitness for use. It is especially suitable for expert/user evaluation workshops or pilot-based assessment.
- **Link:** <a href="https://datascience.codata.org/articles/799/files/submission/proof/799-1-3963-1-10-20180510.pdf" target="_blank" rel="noopener noreferrer">https://datascience.codata.org/articles/799/files/submission/proof/799-1-3963-1-10-20180510.pdf</a>
- **Main assessment use:** Expert- and user-based assessment of ontology usability, fitness for use, and practical adoption
