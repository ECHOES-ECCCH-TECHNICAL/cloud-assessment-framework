# Data usage

| | |
| --- | --- |
| **Primary section** | **Section 6.4 — Evaluation of data usage** |
| **Evaluation basis** | <ul><li>Access, reuse or interaction evidence related to data resources</li><li>Logs and analytics where available and appropriate</li><li>Download, access, query or integration records</li><li>Provider-supplied usage information where usage is not captured directly by ECCCH services</li><li>Pilot or user-study evidence concerning data reuse</li></ul> |
| **Core dimensions / criteria** | <ul><li>Data usage</li></ul> |
| **Main instruments** | <ul><li>Usage analytics</li><li>Access, download or query statistics</li><li>Provider-supplied usage reports</li><li>Pilot evaluation outputs</li><li>Reuse evidence from workflows, services or applications</li></ul> |
| **Expected output** | <ul><li>Evidence of access, uptake or reuse of data resources</li><li>Usage indicators suitable for later aggregation or interpretation</li><li>Identification of resources with limited, emerging or significant reuse</li><li>Evidence supporting monitoring of uptake and value over time</li></ul> |
| **What resource providers should prepare** | <ul><li>Description of how the data resource is accessed</li><li>Available usage, access or download statistics</li><li>Known examples of reuse</li><li>Information on external access channels not visible to ECCCH analytics</li><li>Relevant pilot or user feedback where available</li></ul> |
