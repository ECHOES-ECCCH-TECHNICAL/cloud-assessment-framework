# Summary of evaluation basis, core dimensions and expected outputs

The following summary tables provide a compact overview of how the Cloud Evalution Framework is planned to be applied to
the main resource categories considered in deliverable D9.1. For each category, the tables
identify the main evaluation basis, the core dimensions or relevant criteria, the principal
types of assessment instruments, the expected assessment output, and the evidence
that resource providers or other relevant contributors should prepare. These tables also support navigation across the more detailed resource-specific
subsections of [D9.1 deliverable](https://doi.org/10.5281/zenodo.20448153).

!!! info "A note on prioritization of evaluation dimensions"

    **Core dimensions** are essential for the initial implementation phase.
    **Context-dependent dimensions** are relevant only to certain resource
    categories or use cases. **Not currently prioritized** dimensions, although
    relevant, are not part of the first implementation phase of the framework.
    The tables below currently focus on core dimensions.
