# Evaluation Dimensions

Each resource category can be evaluated across multiple facets or quality dimensions, depending on its nature and intended use. These dimensions may include, among others, data quality, semantic interoperability, software performance and reliability, usability and accessibility, compliance and security, sustainability, and community relevance and impact.  
Not all dimensions apply equally to all resource categories. Their applicability depends on the type of resource under evaluation. These dimensions are specified in the corresponding dedicated sections below. This section documents the evaluation dimensions for each resource category.

- [Data](data_dim.md) — Evaluation dimensions for data-related resources.
- [Metadata](metadata_dim.md) — Evaluation dimensions for metadata-related resources.
- [Ontology](ontology_dim.md) — Evaluation dimensions for ontology-related resources.
- [Software](software_dim.md) — Evaluation dimensions for software-related resources.
- [Digital services](service_dim.md) — Evaluation dimensions for digital services.
